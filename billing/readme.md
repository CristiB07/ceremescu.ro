#Billing
Aplicație de facturare, versiunea 2. Include facturare, chitanțe, facturare abonamente. Facturile și chitanțele se pot salva pdf. Facturile sunt încărcate în efactura. Citește facturile primite în efactura. Se pot marca drept achitate facturile primite, respectiv emise și se face totalurile din conturile bancare. În financedashboard.php este făcută o prezentare a situației financiare - încasări, plăți, facturi emise, facturi primite, comparație cu anul anterior.
